**Cheta (Formerly PioTech)**

***A Borderless, Permissionless Education Platform***

Cheta is a next-generation education platform designed to make knowledge open, decentralized, and accessible to anyone in the world. It combines modern web technology with AI and blockchain infrastructure to build a platform where anyone can teach, learn, and innovate without permission barriers.

Unlike traditional education platforms that restrict content creation, Cheta allows any niche, discipline, or expertise to exist on the platform. If you have knowledge, you can share it.

**Vision**

Education should be:

*Borderless* – accessible from anywhere in the world

*Permissionless* – anyone can create and publish courses

*Trustless* – credentials can be verified through blockchain technology

*Community Driven* – users shape the direction of the platform

Cheta aims to build an ecosystem where learning, innovation, and collaboration are open to everyone.

**Core Features**

*Permissionless Course Creation*

Anyone can upload and publish a course without needing approval from a centralized authority.

*Borderless Learning*

Learners from anywhere in the world can access courses across every niche.

*Blockchain-backed Certificates*

Course completion certificates can be issued and verified using blockchain technology, making them tamper-proof and verifiable.

*AI-powered Learning*

AI tools help personalize the learning experience, improve course discovery, and support learners throughout their journey.

*Community Governance*

Future governance mechanisms will allow the community to influence platform decisions.

*Suggestion-driven Innovation*

Users can propose ideas and innovations for the platform, helping drive new projects and improvements.

**Project Philosophy**

Most education platforms are centralized gatekeepers.

Cheta flips that model by making education:

Open

Permissionless

Creator-driven

Globally accessible

This allows knowledge to emerge from anywhere and anyone, not just institutions.

**Running the Project**

Clone the repository and navigate into the project directory:

```bash
cd piotech
```

Then start the development server:

```bash
python manage.py runserver
```

The application will run locally on:

```bash
http://127.0.0.1:8000/
```
